## Cybersecurity Frameworks

### NIST Cybersecurity Framework (CSF) 2.0

**Source:** [nist.gov/cyberframework](https://www.nist.gov/cyberframework)

The foundation for the Shepherd/vCIO domain. The CSF organizes security into GOVERN, IDENTIFY, PROTECT, DETECT, RESPOND, RECOVER functions.

### NIST SP 800-171 Rev 3

**Source:** [csrc.nist.gov/publications/detail/sp/800-171/rev-3/final](https://csrc.nist.gov/publications/detail/sp/800-171/rev-3/final)

Security requirements for protecting Controlled Unclassified Information (CUI).

**Related Resource:** [NIST SP 800-171 Small Business Primer](https://nvlpubs.nist.gov/nistpubs/SpecialPublications/NIST.SP.1318.pdf) ‚Äî Excellent starting point for smaller organizations.

### NIST SP 800-53 Rev 5

**Source:** [csrc.nist.gov/publications/detail/sp/800-53/rev-5/final](https://csrc.nist.gov/publications/detail/sp/800-53/rev-5/final)

Comprehensive security and privacy controls catalog. Reference for deeper control implementation.

### CIS Controls

**Source:** [cisecurity.org/controls](https://www.cisecurity.org/controls)

Prioritized set of actions for cyber defense. More prescriptive than NIST CSF.

## DevOps & Delivery

### DORA Research

**Source:** [dora.dev](https://dora.dev/)

The definitive research on DevOps practices and organizational performance. Key metrics: Deployment Frequency, Lead Time for Changes, Change Failure Rate, Time to Restore Service.

## Change Management

### Kotter's 8-Step Model

**Source:** [kotterinc.com](https://www.kotterinc.com/methodology/8-steps/)

1. Create urgency
2. Form guiding coalition
3. Create vision
4. Communicate vision
5. Remove obstacles
6. Create quick wins
7. Build on change
8. Anchor in culture

### ADKAR Model (Prosci)

**Source:** [prosci.com/methodology/adkar](https://www.prosci.com/methodology/adkar)

- **A**wareness of need
- **D**esire to participate
- **K**nowledge of how
- **A**bility to implement
- **R**einforcement to sustain

### Bridges' Transition Model

**Source:** William Bridges, "Managing Transitions"

- **Endings** ‚Äî What we're leaving behind
- **Neutral Zone** ‚Äî The messy middle
- **New Beginnings** ‚Äî What we're moving toward
